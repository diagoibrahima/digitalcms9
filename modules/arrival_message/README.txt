INTRODUCTION
------------

The Arrival Message allows site admin to configure the message to display after user login.
This module is basically intended for site-builders who are new to Drupal or just want an easy configuration to set the message after user login.

 * For a full description of the module visit:
   https://www.drupal.org/project/arrival_message

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/arrival_message


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install the arrival_message module as you would normally install a contributed
   Drupal module.
   Visit https://www.drupal.org/node/1897420 for further information.


CONFIGURATION
-------------

  Goto /admin/config/system/site-information and
  set the Arrival Message.


TROUBLESHOOTING
---------------

Clear site cache once.
